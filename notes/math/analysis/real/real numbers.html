What are the real numbers?

\(\text{The real number system (the reals) is a set }\{a,b,c,...\} \text{ on which the operations of addition and mulitplication are}\)

\(\text{defined such that every pair of reals has a unique sum and product, both real numbers, with these properties:}\)

\(\textbf{(A) }a+b=b+a\text{ and }ab=ba\text{ (commutative laws)}\)

\(\textbf{(B) }(a+b)+c=a+(b+c)\text{ and }(ab)c=a(bc)\text{ (associative laws)}\)

\(\textbf{(C) }a(b+c)=ab+ac\text{ (distributive law)}\)

\(\textbf{(D) }\text{There are distinct real bumbers }0\text{ and }1\text{ such that }a+0=a\text{ and }a\cdot1=a\forall{a}\)

\(\textbf{(E) }\text{For each }a\text{ there is a real number }-a\text{ such that }a+(-a)=0\text{, and if }a\neq0\text{, there is a real number }\frac{1}{a}\text{ such that }a\cdot\frac{1}{a}=1\)

\(\text{Regarding properties }\textbf{(A) }\text{ through }\textbf{ (E)}\text{, a set with these properties is called a }\textbf{field}\)

\(\text{Ex. The rational numbers form a field under addition and multiplication.}\)

    • How are the reals ordered?

    \(\text{The real number system is ordered by the relation}<\text{which has the following properties: }\)

    \(\textbf{(F) }\text{For each pair of real numbers }a\text{ and }b\text{, exactly one of the following is true: }a=b\text{, }a{<}b\text{,or }b{<}a\)

    \(\textbf{(G) }\text{The relation}<\text{is }\textbf{transitive}\text{ (If }a\,{<}\,b\text{ and }b\,{<}\,c\text{, then }a\,{<}\,c\text{)}\)

    \(\textbf{(H) }\text{If }a\,{<}\,b\text{, then }a+c\,{<}\,b+c\text{ for any }c\text{ and if }0\,{ < }\,c\text{ then }ac\,{<}\,bc\)

    \(\text{Regarding properties }\textbf{(A) }\text{ through }\textbf{ (H)}\text{, a set with these properties is called an }\textbf{ordered field}\)

    Theorem 1.1: \(\text{If }a\text{ and }b\text{ are two real numbers, then }|a+b|{<}|a|+|b|\) (triangle inequality)

    Correlary 1.1: \(|a-b|\geq||a|-|b||\)

    Correlary 1.2: \(|a+b|\geq||a|-|b||\)

    • The Supremum of a Set

    \(\text{A set }S\text{ of real numbers is }\textbf{bounded above }\text{if there is a real number }b\text{ such that }x\leq{b}\text{ whenever }x\in{S}.\)

    \(\text{This means: }\)

    • \(b\text{ is an upper bound of }S\)
    • \(\text{If some } a>b\text{, }a\text{ is an upper bound of }S \text{ because of property }\textbf{(G)}\text{ (ordered fields are transitive)}\)

    \(\text{If }\beta\text{ is an upper bound of }S\text{, and there exists no number less than }\beta\text{ that is also a lower bound of }S\text{, then }\beta\text{ is a}\textbf{ supremum}\text{ of }S\)

    \(\text{The notation for this is }\beta=\sup{S}\)

    Examples

    \(\text{Consider the example in which }S\text{ is the set of non-negative integers. }\)

    • \(\text{Since all }x\in{S}<{0}\text{, 0 is an upper bound of }S\)
    • \(\text{There are also no upper bounds of }S\text{ that are less than 0 (per the definition of the set)}\)
      • \(\text{If you subtract some number }\epsilon\text{ from 0, there will still be a larger negative number}\)
    • \(\text{Therefore }\sup{S}=0\)
    • \(\text{Now we know that for some set }S\text{, }\sup{S}\text{ isn't nessasarely in }S\)

    \(\text{Now consider the example in which }S\text{ is the set of negative integers}\)

    • \(\text{Since all }x\in{S}<{0}\text{, 0 is an upper bound of }S\)
    • \(\text{However, }-1\text{ is also an upper bound of }S\text{, and }-1<{0}.\)
    • \(\text{And }-2\text{ isn't an upper bound of }S\)
    • \(\text{Therefore }\sup{S}=-1\)

    \(\textbf{(I)}\text{ The Completeness Axiom: If a nonempty set of real numbers is bounded above, then it has a supremum.}\)

    \(\text{This property cannot be derived from properties }\textbf{(A)}\text{ through }\textbf{(H)}\)

    \(\text{Because of completeness, we can say the real nymber system is a }\textbf{complete ordered field}\text{ and in fact the ONLY complete ordered field. }\)

    Theorem 1.2: If a nonempty set \(S\) of real numbers is bounded above, then \(\sup{S}\) is the unique real number \(\beta\) such that

    • \(x\leq\beta\text{ for all }x\in{S}\)
    • \(\text{If }\epsilon>0\text{, there is an }x_0\in{S}\text{ such that }x_0>\beta-\epsilon\)

    \(\text{A set }D\text{ is dense in the reals if every open interval }(a,b)\text{ contains a member of }D.\)

    Theorem 1.4: The rational numbers are dense in the reals

    Examples

    \(\text{The Rational Numbers aren't complete}\)

    \(\text{To show this, suppose we have some set }S=\{x|x\text{ is rational and }x<\pi\}\)

    \(\text{Because of theorem 1.4, there exists some }x_0\text{ such that }\pi-\epsilon<{x_0}<\pi\)

    \(\text{Because of theorem 1.3, that means }\pi=\sup{S}. \text{ Since }\pi\text{ is irrational, that means there exists a rational upper bound of }S, x_1. \)

    \(\text{We know }\pi<{x_1}\implies\text{ there exists some rational number }x_2\text{ such that }\pi<{x_2}<{x_1}\text{ since the rationals are dense in the reals}\)

    \(\text{Since }x_2\text{ is also a rational upper bound of }S\text{, }S\text{ has no rational supremum}\)

    Theorem 1.5: The irrational numbers are dense in the reals

    Infimum of a Set

    \(\text{A set }S\text{ of real numbers is }\textbf{bounded below }\text{if there is a real number }\alpha\text{ such that }x\geq{\alpha}\text{ whenever }x\in{S}.\)

    \(\text{This means: }\)

    • \(\alpha\text{ is a lower bound of }S\)
    • \(\text{If some } a<\alpha\text{, }a\text{ is a lower bound of }S \text{ because of property }\textbf{(G)}\text{ (ordered fields are transitive)}\)

    \(\text{If }\alpha\text{ is a lower bound of }S\text{, and there exists no number greater than }\alpha\text{ that is also a lower bound of }S\text{, then }\alpha\text{ is an}\textbf{ infimum}\text{ of }S\)

    \(\text{The notation for this is }\alpha=\inf{S}\)

    Theorem 1.6: If a nonempty set \(S\) of real numbers is bounded below, then \(\inf{S}\) is the unique real number \(\alpha\) such that

    • \(x\geq\alpha\text{ for all }x\in{S}\)
    • \(\text{If }\epsilon>0\text{, there is an }x_0\in{S}\text{ such that }x_0<\alpha+\epsilon\)

    The Extended Real Numbers

    \(\text{A nonempty set }S\text{ of real numbers is }\textbf{unbounded above }\text{if it doesn't have an upper bound }\)

    \(\text{or }\textbf{unbounded below}\text{ if it doesn't have a lower bound.} \)

    \(\text{We can make up two points, }+\!\infty\text{ and}-\!\infty\text{ to adjoin to the real number system. }\)

    \(\text{We also define the order relationships between them and some }x\in\mathbb{R}\text{ by }-\!\infty<{x}<\infty\)

    \(\infty\text{ and}-\!\infty\text{ are called points at infinity. }S\text{ is unbounded above if }\sup{S}=\infty\text{ and unbounded below if }\inf{S}=-\infty\)

    Arithmetic relationships among \(-\infty,\infty,\) and \(\mathbb{R}\)

    \(\text{If }x\in\mathbb{R}\)

    • \(x+\infty=\infty+x=\infty\)
    • \(x-\infty=-\infty+x=-\infty\)
    • \(\frac{x}{\infty}=\frac{x}{-\infty}=0\)

    \(\text{If }x>0\)

    • \(x\cdot\infty=\infty\cdot{x}=\infty\)
    • \(x\cdot-\infty=-\infty\cdot{x}=-\infty\)

    \(\text{If }x<{0}\)

    • \(x\cdot\infty=\infty\cdot{x}=-\infty\)
    • \(x\cdot-\infty=-\infty\cdot{x}=\infty\)

    \(\text{Also }\)

    • \(\infty+\infty=\infty\cdot\infty=(-\infty)(-\infty)=\infty\)
    • \(-\infty-\infty=\infty(-\infty)=(-\infty)(\infty)=-\infty\)
    • \(|\infty|=|-\!\infty|=\infty\)

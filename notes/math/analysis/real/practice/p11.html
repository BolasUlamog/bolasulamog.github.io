Problem 11

Prove Theorem 1.6: If a nonempty set \(S\) of real numbers is bounded below, then \(\inf{S}\) is the unique real number \(\alpha\) such that

  • \(x\geq\alpha\text{ for all }x\in{S}\)
  • \(\text{If }\epsilon>0\text{, there is an }x_0\in{S}\text{ such that }x_0<\alpha+\epsilon\)
Reveal Answer

\(\text{Let some set }T=\{x\,|\,-x\in{S}\}\)

\(\text{This means }T\text{ is bounded above when }S\text{ is bounded below.}\)

\(\text{By theorem 1.2, }T\text{ has a unique supremum }-\alpha\text{ or }\sup{T}=-\alpha\)

\(\text{Let }x\in{S}\implies-x\leq-\alpha\text{(theorem 1.2) so } x\geq\alpha\)

\(\text{Also, if }\epsilon>0\text{ and }x_0\in{S}\text{ such that } -x_0>-\alpha-\epsilon\text{ (theorem 1.2) }\implies{x}_0<\alpha+\epsilon\)

\(\text{Now we know there exists some }\alpha\text{ that is the infimum of }S\text{ but must prove its unique.}\)

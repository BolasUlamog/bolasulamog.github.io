Math 61 - Week 9

Definition: Function

A function \(f: A \to B\) is a relation such that each element \(a \in A\) is associated with exactly one element \(b \in B\).

Proof / Example

Consider \(f: \mathbb{R} \to \mathbb{R}\) given by \(f(x) = x^2\).

  • Each real number \(x\) has exactly one image \(x^2\).
  • Hence \(f\) is a function.

Theorem: Injective Function

A function \(f: A \to B\) is injective if \(f(a_1) = f(a_2) \implies a_1 = a_2\).

Proof / Example

Example: \(f(x) = 2x + 1\) is injective because if \(2x_1 + 1 = 2x_2 + 1\), then \(x_1 = x_2\).

Exercises

  • Give an example of a surjective function from \(\mathbb{R} \to \mathbb{R}\).
  • Prove that \(f(x) = x^3\) is injective.

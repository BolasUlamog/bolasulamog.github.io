Math 61 — Week 1

Real analysis is built on the structure of fields — specifically the real numbers \(\mathbb{R}\).

The Three Axioms for \(\mathbb{R}\)

  1. Arithmetic: Field axioms
  2. Order: Order axioms
  3. No Holes: Completeness axiom

Fact: The real numbers are the unique field that satisfy all three axioms.

Note that this course also covers linear algebra, but this will be on a seperate page and will be hyperlinked to the bottom of this one.

Definition — Field

\((\mathbb{F}, +, \cdot)\) is called a field if it satisfies the following properties:

Field Axioms (F1–F6) — click to expand
  • F1 — Commutativity:
    For all \(a, b \in \mathbb{F}\), \[ a + b = b + a, \quad ab = ba \]
  • F2 — Associativity:
    For all \(a, b, c \in \mathbb{F}\), \[ (a + b) + c = a + (b + c), \quad (ab)c = a(bc) \]
  • F3 — Distributivity:
    For all \(a, b, c \in \mathbb{F}\), \[ a(b + c) = ab + ac \]
  • F4 — Identities Exist:
    There are special elements \(0, 1 \in \mathbb{F}\) such that \[ a + 0 = a \quad \text{and} \quad a \cdot 1 = a \quad \forall a \in \mathbb{F} \]
  • F5 — Additive Inverse:
    For every \(a \in \mathbb{F}\), there exists \(-a\) such that \[ a + (-a) = 0 \]
  • F6 — Multiplicative Inverse:
    For every \(a \neq 0\), there exists \(a^{-1}\) such that \[ a \cdot a^{-1} = 1 \]

Lemma: Inverses are unique

(If \(a+b=0\) and \(a+c=0\) then \(b=c\))

Proof (click to expand)

\(b=0+b\) (F4)

\(=b+0\) (F2)

\(=b+a+c\)

\(=(b+a)+c\) (F2)

\(=(a+b)+c\) (F1)

\(=0+c\) (F1)

\(=0+c\) (F1)

\(=c\) (F4)

Lemma: \(0\cdot{a}=0\text{ and } (-1)\cdot{a}=-a\text{ for all }a\in\mathbb{F}\)

Proof (click to expand)

Proof of part 1:

\(0\cdot{a}=(0+0)\cdot{a}\) (F4)

\(=a\cdot(0+0)\) (F1)

\(=a\cdot{0}+a\cdot{0}\) (F3)

\(=0\cdot{a}+0\cdot{a}\)

Adding \(-(0\cdot{a})\) to both sides yields

\(0\cdot{a}-0\cdot{a}=0\cdot{a}+0\cdot{a}-0\cdot{a}\)

\(\implies{0=0\cdot{a}}\qquad\blacksquare\)

Proof of part 2:

\(a+(-1)\cdot{a}=1\cdot{a}+(-1)\cdot{a}\) (F4)

\(=(1+(-1))\cdot{a}\) (F1)+(F3)

\(=0\cdot{a}\) (F5)

\(=0\) (Part 1)

Hence by uniqueness of inverses, \(-1\cdot{a}=-a\)

Lecture 2

Order Axioms

\((\mathbb{F},+,\cdot)\) is an ordered if there exists a subset \(\mathbb{P}\subseteq\mathbb{F}\) s.t. the following order axioms hold:

Order Axioms (O1–O2) — click to expand
  • O1 — Exactly one of the following holds for every \(a\in\mathbb{F}\):
    • \(a=0\)
    • \(a\in\mathbb{P}\)
    • \(-a\in\mathbb{P}\)
  • O2:
    For all \(a,b\in\mathbb{P}\), \[ a+b\in\mathbb{P}, \quad a\cdot{b}\in\mathbb{P} \]
Notation: Order Relations

1. Write \(a>0\) for \(a\mathbb{P}\)

2. Write \(a>b\) or \(b<{a}\) if \(a-b=a+(-b)>0\)

3. Write \(a\geq{b}, b\leq{a}\) if (\(a>b\) or \(a=b\))

Lemma: \(1>0\)

Proof (click to expand)

Assume \(1<{0}\) (\(1\neq{0}\) by F4)

\(\implies{(-1)>0}\) by (F4) and (O1)

\((-1)(-1)>0\) by O2

By lemma \((-1)(-1)\) is the inverse of \(-1\)

By another lemma and the fact \(1\) is the inverse of \(-1\) we know \((-1)(-1)=1\)

Hence \(1>0\) which is a contradiction.

Completeness

An ordered field \((\mathbb{F},+,\cdot)\) is complete if any \(S\subseteq\mathbb{F}\) that is bounded above has a least upper bound. From now on, take \(\mathbb{R}\) to be the ONLY complete ordered field.

Definition/Notation: Bounds

1. \[S\subseteq{\mathbb{F}}\text{ is }\left\{\begin{array} \text{\text{bdd above}} \\ \text{bdd below} \end{array}\right\}\text{ if }\exists\alpha\in\mathbb{F}\text{ s.t. }\left\{\begin{array} x\leq\alpha \\ x\geq\alpha \end{array}\right\},\forall{x}\in{S}\]

Such \(\alpha\) is called \[\left\{\begin{array} \text{\text{an upper bound}} \\ \text{ a lower bound} \end{array}\right\}\]

Ex. \((0,1)\in\mathbb{R}\): \(1,15,100^{100}\) upper bounds


  • \(S\subseteq\mathbb{F}\) is bounded if it's bounded above and below
  • \(S\subseteq\mathbb{F}. \beta\in\mathbb{F}\) is a least upper bound if the following both hold:
    • \(\beta\) is an upper bound of \(S\)
    • If \(\alpha\) is an upper bound of \(S\) then \(\beta\leq{\alpha}\). Similarly for the greatest lower bound.
  • Denote the least upper bound of \(S\) as \(\sup{S}\) and the greatest lower bound of \(S\) as \(\inf{S}\).

rmk: \(\sup{S}\) need not be in \(S\)

\(\mathbb{N}, \mathbb{Z}, \mathbb{Q}\)

Definitions (click to expand)

1. The natural numbers are defined as \[ \mathbb{N} = \{1, 1+1, 1+1+1, \ldots\}\subseteq{\mathbb{R}} \]

2. The integers are defined as \[ \mathbb{Z} = \mathbb{N} \cup \{0\} \cup \{-n : n \in \mathbb{N}\}\subseteq{\mathbb{R}} \]

3. The rational numbers are defined as \[ \mathbb{Q} = \left\{\frac{a}{b} : a, b \in \mathbb{Z}, b \neq 0\right\} \]

Prop: Archimedian Principle

This is one of the most fundemental and important ideas in real analysis. There are many ways to write this that are equivilient but we will prove the version that states that \(\mathbb{N}\) is not bounded above. In fact, this is equiviliant to stating that for any two positive real numbers \(a,b\) there exists a natural number \(n\) such that \(a<{n}\cdot{b}\)

Proof (click to expand)

Assume for the sake of contradiction that \(\mathbb{N}\) were bounded above. If it were, because of completeness, then there exists \(\alpha=\sup\mathbb{N}\)

\(\implies{n}\leq{\alpha},\forall{n}\in\mathbb{N}\)

\(\implies{n+1}\leq{\alpha},\forall{n}\in\mathbb{N}\)

\(\implies{n}\leq{\alpha-1}\) which is a contradiction since \(\alpha\) is the sup.

Hence \(\mathbb{N}\) is not bdd above. \(\qquad\blacksquare\)

Sequences

A sequence of real numbers is a map \(\mathbb{N}\to\mathbb{R}\). We denote this by \[\{a_n\}_{n=1}^\infty\text{, where }a_n=a(n)\]

Definition of the limit

Let \(\{a_n\}_{n=1}^\infty\subseteq{\mathbb{R}}\). We say \(\{a_n\}_{n=1}^\infty\) has a limit \(L\in\mathbb{R}\), write \(\lim_{n\to\infty}a_n=L\) if \[\text{ for all }\epsilon>0,\exists{N\in{\mathbb{N}}}\text{ s.t. }|a_n-L|<{\epsilon},\forall{n\geq{N}}\]< /p>

\[\{a_n\}^\infty_{n=1}\subseteq\mathbb{R} \text{ is }\left\{\begin{array}\text{\text{convergent if it has a limit}} \\\text{divergent otherwise}\end{array}\right\}\]

Prop:

If \(\{a_n\}^\infty_{n=1}\subseteq{\mathbb{R}}\) is bdd above and increasing \((a_n)\leq{a_{n+1}}\) then \(\{a_n\}^\infty_{n=1}\) is convergent and is equal to \(\sup{a_n}\)

Proof (click to expand)

Let \(\alpha=\sup{S}\) which we know exists because of the completeness property of the reals.

Now we will prove that there exists \(a_j\) s.t. \(\alpha>a_j>\alpha-\frac{1}{j}\)

Pf: Suppose not for the sake of contradiction. This means that \(\alpha-\frac{1}{j}\) is an upper bound which is a contradiction since \(\alpha\) is the sup. Therefore this is true.

Now we know that since \(\alpha-\frac{1}{j}<{a_j}<\alpha\), \(|\alpha-a_j|<|\alpha-\frac{1}{j}|\)

Now, for all \(\varepsilon>0\), there exists some \(\frac{1}{J}, J\in{\mathbb{N}}\) such that \(1>J\varepsilon\implies\frac{1}{J}>\epsilon\) (Archimedian).

This means that for all \(j>J\) (or \(\frac{1}{j}<\frac{1}{J} \)), \(\lvert{a_j}-\alpha\rvert\leq\frac{1}{j}<\frac{1}{J}<\varepsilon\).

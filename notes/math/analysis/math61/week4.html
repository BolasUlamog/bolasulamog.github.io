Continuity in Multiple Dimensions

Given \(A\subseteq\mathbb{R}^n\), \(f:A\to\mathbb{R}^m\), \(a\in{A}\), we say \(f\) is continuous at \(a\) if for all \(\varepsilon>0\) there exists a \(\delta>0\) s.t. \(\|f(x)-f(a)\|<\varepsilon\) whenever \(\|x-a\|<\delta\) and \(x\in{A}\)

\(f\) is continuuous if it is continuous at all \(a\in{A}\)

Notation and Definitions

1. \(A\subseteq\mathbb{R}^n\). We say \(a\in{A}\) is an isolated point if there exists \(\rho>0\) s.t. \(B_\rho(a)\cap{A}=\{a\}\)

2. \(A\subseteq\mathbb{R}^n,f:A\to\mathbb{R}^m, a\in{A}\) is not an isolated pt. We say \(\lim_{x\to{a}}=L\in\mathbb{R}^m\)

3. \(f\) is continuous at an non-isolated point iff \(\lim_{x\to{a}}f(x)=f(a)\)

We can also say for (3) that if \(A\subseteq\mathbb{R}^n,f:A\to\mathbb{R}^m,a\in{A}\) that \(f\) is continuous at \(A\iff\) for any sequence \(\{a^{(k)}\}^\infty_{n=1}\subseteq{A}\), it holds that \(\lim_{k\to\infty}f(a^{(k)})=f(a)\)

Thm:

If \(A\subseteq\mathbb{R}^m\) compact, \(f:A\to\mathbb{R}\) continuous, then \(f\) is bounded and achieves max and min

Proof (click to expand)

Pf: Suppose not for the sake of contradiction

Then for all \(k\in\mathbb{N}\) there exists \(x^{(k)}\in{A}\) s.t. \(||f(x^{(k)})||\geq{k}\)

Since \(A\) is compact, there exists a subsequence \(\{x^{(k_j)}\}_{j=1}^\infty\) and \(a\in{A}\) s.t. \(\lim_{j\to\infty}x^{(k_j)}=a\)

By continuity, \(\lim_{j\to\infty}f(x^{k_j}=f(a)\) which is convergent and thus bounded which is a contradiction.

Metric Spaces

\((X,d)\) is a metric space if \(X\) is a set and \(d:X\times{X}\to\mathbb{R}\) satisfies

  • (M1): \(d(x,y)\geq0\) for all \(x,y\in{X}\) and ''\(=\)"\(\iff{x=y}\)
  • (M2): \(d(x,y)=d(y,x)\) for all \(x,y\in{X}\)
  • (M3): \(d(x,z)\leq{d(x,y)+d(y,z)}\) for all \(x,y,z\in{X}\)
Examples
  • 1. \[X=\mathbb{R}^n, d(x,y)=||x-y||=\sqrt{\sum_{i=1}^n(x_i-y_i)^2}\]
  • 2. \[X\subseteq{\mathbb{R^n}}, d(x,y)=||x-y||\]
  • 3. \[X\text{ set. }d(x,y)=\binom{1\text{ if }x\neq{y}}{0\text{ if }x=y}\]
  • 4. \[V\text{ vector space over }\mathbb{R},||\cdot||:V\to\mathbb{R}\text{ norm }\]
Balls, open/closed sets, and limit points in metric spaces

1. \(\{x_n\}_{n=1}^\infty\subseteq{X}, \lim_{x\to\infty}x_n=x\in{X}\) if for all \(\varepsilon>0\) there exists \(N\in\mathbb{N}\) s.t. \(d(x_n,x)<\varepsilon\) whenever \(n\geq{N}\)

2. \(y\in{X},\rho>0\). Our ball, \(B_\rho(x)=\{x\in{X}:d(x,y)<\rho\}\)

3. \(A\subseteq{X},x\in{X}\) is a limit point of \(A\) if there exists \(\{a_n\}^\infty_{n=1}\subseteq{A}\) s.t. \(\lim_{n\to\infty}a_n=x\)

4. \(A\subseteq{X}\) is open if for all \(x\in{A}\) there exists \(\rho>0\) s.t. \(B_\rho(a)\leq{A}\)

5. \(A\subseteq{X}\) is closed if it contains all its limit points.

6. \(A\subseteq{X}\) is bounded if there exists \(y\in{X}, \rho>0\) s.t. \(A\subseteq{B_\rho(y)}\)

7. \(A\subseteq{X}\) is compact if for any \(\{a_n\}^\infty_{n=1}\subseteq{A}\) there exists \(\{a_{n_k}\}^\infty_{k=1}\) and \(a\in{A}\) s.t. \(\lim_{k\to\infty}a_{n_k}=a\)

Prop:

1. \(X,\phi\) open and closed, respectively

2. \[\binom{\text{Union}}{\text{Intersection}}\text{ of }\binom{X}{\phi}\text{ is }\binom{\text{open}}{\text{closed}}\]

3. \[\text{Finite }\binom{\text{Intersection}}{\text{Union}}\text{ of }\binom{X}{\phi}\text{ is }\binom{\text{open}}{\text{closed}}\]

Thm:

\((X,d)\) metric spaces and \(A\subseteq{X}\). If \(A\) is compact then \(A\) is closed and bounded

Proof (click to expand)

Pf (boundedness): Suppose not for the sake of contradiction

This means for all \(y\) s.t. \(B_\rho(y)\supseteq{A}\)

This means there exists \(a_n\in{A}\) but \(a_n\notin{B_n}(y)\) (ie \(d(a_n,y)\geq{n}\))

However, by compactness, there exists some subsequence \(\lim_{k\to\infty}a_{n_k}=a\). Then by prop, \(\{a_{n_k}\}^\infty_{k=1}\) is bounded but \(d(a_{n_k})\geq{n_k}\geq{k}\) Contradiction!

Pf: Closedness:

Suppose \(x\in{X}\) is a limit point of \(A\). We WTS \(x\in{A}\)

By def, there exists \(\{x_n\}^\infty_{n=1}\subset{A}\) s.t. \(\lim_{n\to\infty}a_n=x\)

By compactness, there exists \(\{a_{n_k}\}^\infty_{n=1}\) s.t. \(\lim{a_{n_k}}=a\in{A}\)

Def Continuity:

Given \((X,d_x),(Y,d_y)\) metric spaces and \(f:X\to{Y}\)

  • 1. Given \(c\in{x}\) we say \(f\) is continuous at \(c\) if for all \(\varepsilon>0\) there exists \(\delta>0\) s.t. \(d_Y(f(x),f(c))<\varepsilon\) whenever \(d_X(x,c)<\delta\) and \(x\in{X}\)
  • 2. \(f\) is continuous if its continuous at all points

Lemma:

\(f\) is continuous at \(c\iff\) for any \(\{x_n\}^\infty_{n=1}\subseteq{X},\lim_{n\to\infty}x_n=c\), it holds that \(\lim_{n\to\infty}f(x_n)=f(c)\)

Proof (click to expand)

Pf: Same as before

Thm:

\((X,d_X),(Y,d_Y),X\text{ compact}, f:X\to{y}\) continuous. Then \(f(x)=\{f(x):x\in{X}\}\) is also compact :D

Proof (click to expand)

Pf: Let \(\{y_n\}^\infty_{n=1}\subseteq{f(X)}\). There there exists \(x_n\in{X}\) s.t. \(y_n=f(x_n)\)

By compactness of \(X\), there exists a subsequence \(\{x_{n_k}\}^\infty_{k=1}\) and \(x\in{x}\) s.t. \(\lim{x_{n_k}}=x\)

By continuity of \(f\) and lemma, \(\lim{y_{n_k}}=\lim{f(x_{n_k})}=f(x)\in{f(X)}\)

Cor:

Given \((X,d_X)\) compact. \(f:X\to\mathbb{R}\) is continuous then

  • 1. \(f(x)\) is odd
  • 2. \(\exists{c_1,c_2}\in{X}\) s.t. \(f(c_1)\leq{f(x)}\leq{f(c_2)}\)

Limits

Similar to the definition in single-variable calculus, we can define the limit to be the following:

\(\lim_{(x,y)\to(a,b)}f(x)=L\implies\) \(\text{ for every number }\epsilon>0\text{, there is some }\delta>0\text{ s.t. if } (x,y)\in{D},\text{ and }0<\sqrt{(x-a)^2-(y-b)^2}<\delta\text{ then }\vert{f(x,y)-L}<\epsilon\vert\)

So how do we show that a given limit doesn't exist?

If a limit exists, it has to have the same value from all possible different paths. Consider the following limits that DON'T exist:
